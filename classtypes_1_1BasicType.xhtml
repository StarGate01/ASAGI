template<typename T >
void types::BasicType< T >::convertByte ( const void *  data,
void *  buf 
) [inline, virtual]

Conversation functions: The value is copied from data to buf, doing transformations between basic types.
E.g.: If the type of this class is "float" and the "convertDouble" is called, *data should be a float and *buf a double.
All these functions have the type "converter_t". This way it is possible to pass them arround as pointers.
