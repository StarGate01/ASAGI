Detailed Description

Stores one grid. It maintains a directory where the current location of blocks is stored. Blocks are only loaded dynamically when required.

unsigned long* grid::DistCacheGrid::m_dictionary [private]

Information about which blocks are currently stored on which node.
Structure:
This array consists of getLocalBlockCount() elements. Each element stores information about one block. It is a list of node/offset pairs which indicate where the block is currently stored. The length of the list is determind by m_dictEntries. The actual length of the list is stored in the element before the list.
