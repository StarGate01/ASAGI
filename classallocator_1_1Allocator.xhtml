template<typename T>
class allocator::Allocator< T >

Pure virtual base class that defines all functions provided by allocators

Template Parameters:
TThe type that is allocated
Warning:
Subclasses can not have a destructor, because this class does not have a virtual one. A virtual constructor here leads to crashes on some systems.
