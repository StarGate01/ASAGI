long cache::CacheManager::get ( unsigned long  blockId,
unsigned long &  cacheOffset,
unsigned char *&  data 
)
inline

Get the cache entry for the the block id. This function will lock the cache entry for further access. Use unlock to unlock the cache entry.

Parameters
blockIdThe requested block id
[out]cacheOffsetOffset (in blocks) in the cache; can be passed to unlock to unlock the cache entry
[out]dataA pointer to the request cache entry
Returns
The block id of the element that is currently stored at the cache position. If this is not equal to blockId the cache entry needs to be filled first. -1 is returned if the cache entry needs to be filled but it is currently empty/does not contain any valid block.
